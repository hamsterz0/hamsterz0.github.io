Technical Experience

ROVER SOCIETY

I am currently the controls lead of the ROVER Society. Me and my teammates developed a Python Code using Pygame to control the movement of the R.O.V.E.R via a Joystick Controller. The values generated by the Python code where passed to the Arduino through a serial port that we opened using 2 Right Hand Circular Polarized Antenna's and the values were send at a rate of 915 MHz. I also made an overlay Python code that helped to parse the GPS values received from the GPS located on the ROVER and send it to an offline map viewer program called GMapCatcher.
